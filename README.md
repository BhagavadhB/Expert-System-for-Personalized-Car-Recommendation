# Expert-System-for-Personalized-Car-Recommendation
#Expert System for Personalized Car Recommendation

# Overview
This project is an **Expert System** built with **Streamlit**, designed to recommend cars based on user-defined preferences such as **budget, fuel type, body type, comfort, safety, and performance priorities**.  
It uses a **weighted scoring algorithm** to evaluate and rank vehicles from the dataset, helping users make data-driven car purchase decisions.

#Features
- Interactive **Streamlit web interface** with a modern, polished UI.  
- **Primary filters:** Budget range, seating capacity, fuel type, and body type.  
- **Soft & Hard filtering** modes for flexible preferences.  
- **Weighted scoring system** based on user priorities:
  - Performance  
  - Economy  
  - Safety  
  - Comfort  
  - Ownership cost  
  - Price sensitivity  
- **Shortlist and Compare** options to evaluate multiple cars side-by-side.  
- Uses **real-world car data** compiled from trusted automotive sources.


#How to Run
1. Ensure Python and all dependencies are installed.  
2. Place all project files ('app.py', 'expert_system.py', and the dataset) in the same folder.  
3. Open a terminal or command prompt in that project folder.  
4. Run the application using the command below:

   streamlit run app.py
